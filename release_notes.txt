## SmartWaste AI V2.0.0

### What's New
- AI-powered waste bin monitoring with Gemini Vision
- Optimized collection routes with OR-Tools
- Real-time GPS tracking for collectors
- Citizen waste reporting with photo upload
- SHG plastic collection management
- Recycler marketplace with bidding system
- Cloudinary image storage
- Production deployment on Render + Vercel

### Download
Install the APK on any Android device (Android 8.0+)

### How to Install
1. Download SmartWasteAI-v2.0.0.apk
2. Enable 'Install from unknown sources' in Android settings
3. Open the APK file and install
